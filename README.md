# 导航栏使用说明

## 功能介绍

本导航栏提供了以下功能：

1. 按分类浏览和访问网站
2. 添加、删除自定义网站
3. 搜索功能（本地搜索和外部搜索引擎）
4. 批量导入网站（JSON格式）
5. 导出网站数据（JSON格式）

## 导入功能使用方法

### 准备JSON文件

1. 创建一个格式正确的JSON文件，参考以下结构：

```json
{
  "study": [
    {
      "name": "网站名称",
      "url": "网站URL"
    },
    {
      "name": "另一个网站名称",
      "url": "另一个网站URL"
    }
  ],
  "life": [
    // 生活分类下的网站
  ],
  "tools": [
    // 工具分类下的网站
  ],
  "web": [
    // 常用外站分类下的网站
  ],
  "campus": [
    // 校园网站分类下的网站
  ],
  "ai": [
    // AI工具分类下的网站
  ]
}
```

2. 确保所有必需字段完整（`name` 和 `url`）
3. 可以只包含部分分类，未包含的分类将保持不变

### 执行导入

1. 点击导航栏顶部的「导入」按钮
2. 在弹出的文件选择器中，选择您准备好的JSON文件
3. 系统将自动验证文件格式，并将新网站合并到您现有的数据中
4. 导入成功后，导航栏将自动刷新显示导入的网站

### 注意事项

- 系统会自动检测并跳过重复的网站（基于URL判断）
- 请确保JSON文件格式正确，否则导入将失败
- 导入不会删除现有网站，只会添加新的网站

## 导出功能使用方法

1. 点击导航栏顶部的「导出」按钮
2. 系统将自动生成一个包含当前所有网站数据的JSON文件
3. 文件名格式为：`导航栏数据_日期.json`，其中日期格式为`YYYY-MM-DD`
4. 文件将自动下载到您的默认下载目录

## 示例文件

项目中包含一个示例导入文件 `example_import.json`，您可以参考此文件创建自己的导入数据。
